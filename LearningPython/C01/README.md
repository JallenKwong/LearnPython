## 人们为何使用Python ##

1. Python更注重可读性、一致性和软件质量。
2. 能提高开发者效率。
3. 高可移植性，跨系统运用。
4. 支持标准库（内置许多功能模块）。
5. Python可通过灵活的集成机制轻松地与应用程序的其他部分进行通信。
6. Python编程丰富有趣。
